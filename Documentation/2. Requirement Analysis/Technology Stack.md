# Prerequisites for Full-Stack Food Ordering App (MERN)

To develop a full-stack food ordering application using **React.js, Node.js, and MongoDB**, the following prerequisites should be considered:

---

## 1. Node.js and npm
- Install **Node.js**, which includes **npm (Node Package Manager)**, on your development machine.  
- Node.js is required to run JavaScript on the server side.  
- **Download:** [Node.js](https://nodejs.org/en/download/)  
- **Installation Instructions:** [Node.js Package Manager Guide](https://nodejs.org/en/download/package-manager/)  

---

## 2. MongoDB
- Set up a **MongoDB database** to store restaurant and booking information.  
- Install MongoDB locally or use a cloud-based MongoDB service.  
- **Download:** [MongoDB Community Edition](https://www.mongodb.com/try/download/community)  
- **Installation Instructions:** [MongoDB Manual](https://docs.mongodb.com/manual/installation/)  

---

## 3. Express.js
- Express.js is a web application framework for Node.js.  
- It handles server-side routing, middleware, and API development.  
- **Installation Command:**  
  ```bash
  npm install express
