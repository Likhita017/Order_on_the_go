## Tools and Technologies Used
- **Frontend**: React
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Deployment**: Heroku

## Database Collections Overview
The application uses the following collections in the MongoDB database:
1. **Users**: Stores user information and authentication data.
2. **Orders**: Holds information about customer orders.
3. **Products**: Contains details of products available for purchase.

## Environment Configuration Details
To properly configure the environment for the application:
1. Make sure to create a `.env` file in the root directory.
2. Add the following variables:
   - `DB_URI`= The URI for connecting to the MongoDB database.
   - `PORT`= The port number the application will run on.
